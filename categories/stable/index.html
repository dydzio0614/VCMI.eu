VCMI 0.99 released

November 1, 2016

2 minute read

We’re back with new major release! A lot of work has been done to ensure game stability. A number of “impossible” crash bugs were tracked down and fixed. This opens up chance for better multiplayer support in the future. RMG has undergone huge improvement, it is now not only closely resembling OH3, but also works much faster. Bonus is about 150 random map templates ported from original game and fan favourites.

VCMI 0.98 released

April 1, 2015

2 minute read

We’re happy to give you new version VCMI 0.98. Previous release 0.97 was not very satisfactory, it has a number of issues with AI as well as crashbugs. Luckily now all these problems are solved and you can play new VCMI 0.98 with no problems. Finally all the spells in game are supported. New additions include missing adventure spells and Earthquake. An important new feature is World View option. You will also notice nice fading effect on picked treasures and adventure map transitions.
